# KStart-Sites

奇趣起始页的用户数据列表

## 如何使用

在起始页中生成 JSON 文件，或按照本项目里的 **示例文件** 进行编写后提交到本项目，在起始页后添加 `?u=名称` 即可访问你的专属起始页。

> 注意：请不要随意修改其他人的已有配置文件，否则将不予通过！
